#### Практическая работа «Жёсткие ссылки»

Необходимо написать консольное приложение на C/C++ с использованием любой библиотеки, но без вызова внешних приложений, которое:

1. Обходит весь указанный каталог и составляет список файлов в нём.
2. Для каждого файла вычисляет любой вид хэша (можно SHA1).
3. Для всех файлов, у которых обнаружен совпадающий хэш, требуется оставить только один экзэмпляр, а остальные нужно заменить на жёсткие ссылки, указывающие на тот экземпляр, который был оставлен.

---

##### Run

all:

```bash
make
```

clean:

```bash
make clean
```
